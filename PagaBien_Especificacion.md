# PagaBien

## Especificación Funcional

**PagaBien** es una aplicación tipo PWA para administrar ingresos,
gastos, cuentas por pagar y servicios realizados, con funcionamiento sin
conexión y sincronización con Google Sheets.

## Tecnologías

-   React + Vite (PWA)
-   NestJS
-   PostgreSQL + SQLite
-   Google Sheets API

## Módulos

-   Dashboard
-   Ingresos
-   Gastos
-   Cuentas por pagar
-   Servicios realizados
-   Clientes
-   Reportes
-   Configuración

## Diseño uniforme

Los módulos Ingresos, Gastos y Cuentas por pagar compartirán la misma
interfaz con búsqueda, filtros, tabla de registros y acciones (Nuevo,
Editar, Eliminar, Exportar, Sincronizar e Imprimir).

## Dashboard

-   Saldo disponible
-   Ingresos y gastos del mes
-   Cuentas por pagar
-   Servicios pendientes
-   Gráficos

## Google Sheets

-   Importación y exportación
-   Sincronización manual y automática
-   Sincronización bidireccional

## Funciones

-   Recordatorios
-   Adjuntar comprobantes
-   PDF/Excel
-   Copias de seguridad
-   PIN/Biometría
-   Offline

## Visión

Integración futura como módulo financiero de un ERP.
